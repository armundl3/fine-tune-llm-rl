# Lesson 1: Supervised Fine-Tuning

### Conceptual Goal
Grasp the foundational theory behind adapting pre-trained models for specialized tasks.

### Key Concepts
- Pre-training vs. Fine-Tuning
- Supervised Fine-Tuning (SFT)
- Instruction Following & Use Cases

### Exercise Overview
This lesson contains practical exercises for creating SFT datasets and understanding the fine-tuning process.

### Folder Structure
- exercises/starter/: Starting code for students
- exercises/solution/: Completed solutions
- exercises/demo/: Instructor demonstrations
