# Exercise: Crafting a Clinical SFT Dataset

### Objective
Understand the structure of an SFT dataset by applying the demonstrated principles to the healthcare scenario.

### Task
Given the inclusion/exclusion criteria for a fictional clinical trial, create 5-10 (patient_summary, eligibility_status) pairs for both eligible and ineligible synthetic patients.

### Starter Code
This file provides the basic structure for creating SFT training data.
