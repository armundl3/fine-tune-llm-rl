# Exercise: SFT for Eligibility Prediction

### Objective
Gain hands-on experience with the SFT training loop on the project's core task.

### Task
Use the provided dataset of (patient_summary, eligibility_status) pairs to fine-tune a small Gemma variant, focusing on configuring the LoRA parameters and analyzing the validation loss.

### Project Mapping
This lab is a direct, guided implementation of Phase 1: Supervised Fine-Tuning.
