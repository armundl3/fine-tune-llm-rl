# Exercise: Creating Clinical Preference Pairs

### Objective
Practice creating preference data by evaluating agent reasoning traces based on patient safety.

### Task
Given a patient case and three agent-generated traces, rank the traces and create the (chosen, rejected) pairs, justifying choices based on the project's goal of prioritizing patient safety.

### Focus Areas
- Trace quality evaluation
- Safety-focused ranking
- Preference pair creation
- Justification reasoning
